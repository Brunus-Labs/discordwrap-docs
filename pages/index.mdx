# Welcome to Discordwrap
 
Discordwrap is a simple, lightweight, and functional API wrapper for Discord.

**Features:**
- Synchronous API requests
- Optionally Async via `async`/`await`
- Full rate limit handling
- No dependencies
- Purely Functional design
- Extremely lightweight

## Getting Started

First time? No problem! Check out our [Quickstart](/quickstart), or if you want to jump in the deep end, check out our [API Reference](/api_reference)

## Use Cases
- Ping a discord server when a script is done running
- Get information about a discord channel for integration with another service
- More to come!
